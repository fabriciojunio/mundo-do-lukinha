# LGPD — Conformidade

## Base Legal
Art. 14 LGPD: dados de crianças requerem consentimento específico do responsável.

## Dados Coletados (localStorage apenas)
| Dado | Finalidade |
|------|-----------|
| Nome/apelido | Personalização do perfil |
| Idade | Adaptar conteúdo à faixa etária |
| Progresso | Salvar XP, moedas, conquistas |
| Histórico | Dashboard do responsável |

## Dados NUNCA Coletados
CPF, RG, endereço, escola, fotos, GPS, cookies de terceiros, dados biométricos.

## Direitos Implementados
- **Acesso:** Área dos Pais mostra todos os dados
- **Portabilidade:** Botão "Exportar Dados" (JSON)
- **Eliminação:** Botão "Excluir Tudo" com confirmação
- **Minimização:** Só o estritamente necessário
- **Transparência:** Aba "Privacidade" explica tudo
